╔════════════════════════════════════════════════════════════════════════════╗
║                           ARCHITECTURE FLOW                                ║
╚════════════════════════════════════════════════════════════════════════════╝

    FRONTEND (Browser)              BACKEND (Flask Server)      DATABASE
    ─────────────────               ──────────────────────      ────────
    
    1. User Signs Up
       │
       ├─→ Firebase Auth
       │   │
       │   └─→ Returns ID Token
       │
    2. Frontend sends Token to Backend
       │
       ├─→ POST /api/goals/set
       │   │
       │   └─→ @require_auth decorator
       │       │
       │       └─→ Verify token ─→ Get uid
       │           │
       │           └─→ Save to Firestore ─→ users/{uid}/goals/
       │
    3. User logs meal
       │
       ├─→ POST /api/meals/add
       │   │
       │   └─→ Meal model saves to:
       │       users/{uid}/meals/{date}/entries/
       │
    4. Dashboard loads
       │
       ├─→ GET /api/summary/today
       │   │
       │   └─→ Meal model calculates totals
       │
       ├─→ GET /api/recommendations/today
       │   │
       │   └─→ RecommendationEngine analyzes data
       │       │
       │       └─→ Returns suggestions & alerts
       │
       └─→ UI updates with live data

╔════════════════════════════════════════════════════════════════════════════╗
║                           API ENDPOINTS REFERENCE                          ║
╚════════════════════════════════════════════════════════════════════════════╝

AUTHENTICATION
    POST   /api/auth/signup            Create account
    POST   /api/auth/login             User login
    POST   /api/auth/verify-token      Verify token
    GET    /api/auth/user-info         Get user info (auth required)
    POST   /api/auth/logout            Logout

GOALS MANAGEMENT
    POST   /api/goals/set              Set goals (auth required)
    GET    /api/goals/get              Get goals (auth required)
    PUT    /api/goals/update           Update goals (auth required)

MEAL TRACKING
    POST   /api/meals/add              Add meal (auth required)
    GET    /api/meals/daily/<date>     Get meals (auth required)
    GET    /api/meals/daily-totals/<date> Get totals (auth required)
    DELETE /api/meals/delete/<date>/<id>  Delete meal (auth required)

RECOMMENDATIONS
    GET    /api/recommendations/today           Today's recs (auth required)
    GET    /api/recommendations/daily/<date>    Date recs (auth required)
    GET    /api/recommendations/weekly-trends   Weekly trends (auth required)
    GET    /api/recommendations/food-database   Food DB (public)

SUMMARY/DASHBOARD
    GET    /api/summary/today          Today's summary (auth required)
    GET    /api/summary/date/<date>    Date summary (auth required)
    GET    /api/summary/weekly         Weekly summary (auth required)
